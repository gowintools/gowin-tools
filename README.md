# GoWin Tools

**Free, private, browser-based tools for everyday work.**

[https://gowin.tools](https://gowin.tools)

---

## What is GoWin Tools?

GoWin Tools is a growing collection of 80+ free utilities — text tools, calculators, developer helpers, business document generators, and more. Everything runs entirely in your browser. No login. No tracking. No data sent to any server.

---

## Features

- **80+ tools** across text, math, developer, finance, and business categories
- **Privacy-first** — all processing happens locally in your browser
- **No login required** — open a tool and use it
- **No ads** — clean, distraction-free interface
- **Chaining** — send output from one tool directly into another
- **Works offline** — once loaded, most tools work without a connection
- **Mobile-friendly** — responsive layout across all screen sizes

## Privacy by Design

GoWin Tools is built on a simple principle: **your data never leaves your device.**

- No form submissions sent to a server
- No user data stored anywhere
- No third-party tracking scripts
- Aggregated, anonymous usage stats only (page views — no personal data)

Read more in [`docs/privacy-first-design.md`](docs/privacy-first-design.md).
