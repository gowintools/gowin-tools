# Sample Data

This folder contains sample files for testing GoWin Tools utilities.

> **Sample Data — For Testing Only**
> These files contain fictional data and are not representative of any real person, business, or transaction.

## Files

| File | Purpose |
|---|---|
| `sample-data.json` | Generic JSON object for testing JSON formatter, converter, and validator tools |

## Using with GoWin Tools

1. Open the relevant tool at [gowin.tools](https://gowin.tools)
2. Paste or upload the sample file
3. Explore the output

All processing happens in your browser — the sample files are never uploaded to any server.
