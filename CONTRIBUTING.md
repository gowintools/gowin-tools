# Contributing to GoWin Tools

Thank you for your interest in GoWin Tools.

## This Repository

This is the **public reference repository** for GoWin Tools. The production codebase is maintained in a private repository. As a result, pull requests here are limited in scope.

## What You Can Do

### Report a Bug
Open a [GitHub Issue](../../issues) with:
- The tool name and URL
- What you expected to happen
- What actually happened
- Browser and OS (if relevant)

### Suggest a Feature or New Tool
Open a [GitHub Issue](../../issues) with the label `suggestion`. Describe:
- The tool or feature you'd like
- Who would benefit from it
- Any similar tools you've seen elsewhere

### Improve the Docs
Docs in the `docs/` folder are editable. If you spot an error or want to clarify something, open a PR — doc improvements are welcome.

### Report a Security Issue
See [SECURITY.md](SECURITY.md) for responsible disclosure guidelines.

## What This Repo Is Not

- A place to request integration with other services (GoWin Tools is intentionally offline-first)
- A place to request accounts, logins, or data storage features

## Code of Conduct

Be respectful. Issues or PRs that are rude, spammy, or off-topic will be closed.
